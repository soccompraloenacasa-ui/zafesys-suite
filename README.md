# ZAFESYS Suite

Smart lock installation management system with admin dashboard and technician PWA.

## Features
- Lead management with Kanban board
- Customer management
- Installation scheduling with calendar
- Technician assignments
- Distributor sales tracking with charts
- Inventory management
- Real-time dashboard

## Stack
- Frontend: React 19 + TypeScript + Tailwind CSS v4
- Backend: FastAPI + SQLAlchemy
- Database: PostgreSQL
- Charts: Recharts

## Deploy
- Frontend: Vercel
- Backend: Railway
